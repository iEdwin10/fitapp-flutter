# FitApp Flutter

A fitness mobile app built with Flutter (Dart) for iOS & Android.

## Screens

- **Onboarding** — Welcome screen with lime green background and CTA button
- **Home** — Dashboard with popular workouts carousel and today's plan
- **Workout Detail** — Dark screen with rounds list and "Lets Workout" CTA

## Design

- Primary color: `#B5E72E` (lime green)
- Dark background: `#1E1E1E`
- Font: Poppins

## Getting Started

```bash
flutter pub get
flutter run
```

## Requirements

- Flutter SDK >= 3.0.0
- Dart SDK >= 3.0.0
